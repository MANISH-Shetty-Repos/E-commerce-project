#  E‑Commerce Web Application

A **full‑stack e‑commerce platform** built using **Java, Spring Boot, Hibernate, MySQL, and JSP**, featuring secure authentication, role‑based access, and scalable backend architecture.

---

##  Project Overview

This E‑Commerce Web Application enables users to **browse products, place orders, and manage accounts**, while administrators can **manage products, users, and orders** through a secured backend system.

The application is developed using **Spring MVC architecture** and follows **Service and DAO design patterns** for clean separation of concerns and maintainability.

---

##  Key Features

### 👤 User Features

* User registration and login
* Secure authentication with Spring Security
* Browse and search products
* Add products to cart
* Place and manage orders

### 🛠️ Admin Features

* Admin login with role‑based access
* Product management (Add / Update / Delete)
* User and order management
* Secure access to admin dashboards

### 🔐 Security

* Spring Security integration
* Role‑based authorization (USER / ADMIN)
* Session management and access control
